User
1. 회원가입 (유효성검사)
2. 로그인
3. 회원정보 수정 (사진, 닉네임)
4. 회원정보 수정 (비밀번호)
5. 회원탈퇴 (게시글, 댓글 사라지지 않음)

Post
1. 게시글 작성 (연타방지)
2. 게시글 조회
3. 게시글 수정 (작성자만 가능)
4. 게시글 삭제 (작성자만 가능) (댓글도 모두 삭제)

Comment
1. 댓글 작성 (연타방지)
2. 댓글 조회
3. 댓글 수정 (작성자만 가능)
4. 댓글 삭제 (작성자만 가능)

추가 디테일
1. 개발자 데이트 입력
2. 유효성 검사 색상 (모두 통과 시 Green으로 변경)
3. 헤더 누르면 게시글 목록 조회 페이지로 넘어감
4. 로그인하지 않으면 로그인 드롭다운이 나옴
5. 로그인하지 않으면 게시글 작성과 댓글 작성이 불과 (게시글과 댓글 조회만 가능)
6. 게시글, 댓글, 닉네임은 공백과 공백 특수문자만으로 입력 불가능
7. 게시글과 댓글 연타로 중복 등록 불가능
8. 게시글과 댓글에서 엔터 그대로 표시됨
9. 개발자와 글쓴이, 자신이 게시글과 댓글 상에서 다르게 표시됨 (색상도 다름!)
10. 알림은 토스트 메세지로 모두 변경
11. 조회수와 댓글수 업데이트 기능
12. 회원 정보 수정시 현재 닉네임인 경우 현재 닉네임이라고 표현
13. 닉네임과 이메일 중복 검사 실시간으로 진행
14. 게시글 작성시 사진 크기 그대로 들어감
15. 게시글 작성시 사진을 넣지 않으면 글만 들어감
16. 게시글, 유저, 댓글 인덱스 구현(DB처럼!)
